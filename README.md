## 🏆 Things I've Built & Shipped (Real Projects)

### 🚀 **Ansible + GitHub Actions Integration**
Built a complete automation framework showing how to orchestrate Ansible playbooks through GitHub Actions workflows. Created example deployments for web servers and local testing with proper secrets management and GitHub Pages documentation. _([AAP Project](https://github.com/cdm227/AAP))_

### 🤖 **GitHub Issue Complexity Analyzer (Copilot SDK)**
Architected an AI-powered issue triage tool during a livestream demo that autonomously fetches GitHub issues, explores repository structure, reads source code, and produces structured complexity assessments using the Copilot SDK. Built with Python, FastAPI, Server-Sent Events, and agentic tool calling. Includes dev container setup, reference implementations, and presenter resources. _([copilot-sdk-github-issue-analyser](https://github.com/cdm227/copilot-sdk-github-issue-analyser))_

### 🏨 **AI Travel Assistant with MCP Servers**
Led development of an end-to-end AI assistant for travel bookings combining Next.js frontend (TypeScript + CopilotKit), .NET backend with Model Context Protocol servers, and Azure Cosmos DB integrations. Implemented semantic flight search with GitHub Models embeddings and secure API key authentication. _([aigenius-maf-travel-assistant](https://github.com/cdm227/aigenius-maf-travel-assistant))_

### 📚 **DevOps Learning Resources**
Curated and created real-world DevOps projects covering AWS 3-tier architecture deployments, scalable VPC designs, CI/CD pipelines, Terraform IaC, Kubernetes orchestration, and monitoring solutions. Projects span beginner → advanced with hands-on labs and production-ready patterns. _([DevOps-Projects](https://github.com/cdm227/DevOps-Projects))_

### 🔄 **CI/CD Training Environments**
Built multiple train-schedule sample apps for Git, Gradle, and Jenkins pipeline exercises—used as learning tools for CI/CD fundamentals including Git workflows, build automation, and pipeline design.

### 🧠 **Infrastructure Automation & Learning**
